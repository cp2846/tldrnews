• Tl;DrNews

About

Tl;DrNews is a site which automatically summarizes articles from around the web and makes those summaries available in a single newsfeed. Tl;DrNews is not affiliated with any of the news sources presented. All content belongs to their respective owners.

Tl;DrNews is powered by NewsApi.

API

In addition, Tl;DrNews provides an API of its own, which can be accessed at /api. The API supports searching summaries by content, source, and date range. See examples of usage below.

endpoint: /api/top
parameters:
    - sources: Comma-separated list of sources to pull from
    - search: String to search in the article title and summary
    - timeBefore: filter for articles before this date (in ISO format)
    - timeAfter: filter for articles after this date (in ISO format)
                       
examples:
    /api/top?sources=engadget,ars technica&search=drone&timeAfter=2018-09-06
    /api/top?sources=cbs news,abc news,techradar&timeAfter=2018-09-06
    
returns:
    [
        {
            title: Title of the article
            source: Source of the article
            url: Url to the article
            imageUrl: Url to the image
            summary: Summary of the article
            date: Date (in ISO format)
        }
    ]
    
endpoint: /api/search/[searchterm]
parameters:
    - [searchterm]: String to search for in the title/summary
   
                       
examples:
    /api/search/3D printing
